# YouTube Extractor

**Analysez vos videos YouTube en quelques clics.**

Extractions, transcriptions, suggestions IA — le tout depuis une seule application.

[![Telecharger](https://img.shields.io/badge/Telecharger-v0.2.1--beta-blue?style=for-the-badge&logo=windows)](https://github.com/NezeCorp/youtube-extractor-releases/releases/latest)

</div>

---

## C'est quoi ?

YouTube Extractor est une application Windows qui permet de recuperer toutes les informations d'une video ou d'une playlist YouTube : titres, descriptions, tags, miniatures, statistiques et transcriptions.

Une intelligence artificielle integree peut ensuite analyser ces donnees et proposer des ameliorations : nouveaux titres, descriptions optimisees, tags pertinents et resumes.

L'IA fonctionne directement sur votre PC, sans envoyer vos donnees sur internet.

---

## Ce que vous pouvez faire

- **Analyser une video ou une playlist entiere** en collant simplement le lien YouTube
- **Recuperer les transcriptions** automatiquement (sous-titres)
- **Obtenir des suggestions IA** pour ameliorer vos titres, descriptions et tags
- **Comparer vos contenus** avec les propositions de l'IA cote a cote
- **Organiser vos analyses** par projet
- **Exporter vos donnees** en CSV, JSON ou Markdown
- **Suivre vos statistiques** : nombre de videos analysees, createurs, historique

---

## L'assistant IA

L'application embarque un modele d'IA (Qwen3-1.7B) qui tourne entierement sur votre machine.

- **Aucune connexion internet requise** pour l'IA
- **Aucune cle API necessaire**
- **Vos donnees restent privees**

Si vous preferez, vous pouvez aussi connecter un service IA externe : Google Gemini, OpenAI (ChatGPT), Groq, Anthropic (Claude), Mistral AI ou Ollama.

## Installation

1. **Telecharger** l'installeur depuis la page [Releases](https://github.com/NezeCorp/youtube-extractor-releases/releases/latest)
2. **Lancer** le fichier .exe telecharge
3. **Patienter** pendant l'installation (le modele IA de ~1.2 Go sera telecharge automatiquement)
4. **C'est pret** — lancez YouTube Extractor depuis le menu Demarrer

> **Note :** Windows SmartScreen peut afficher un avertissement car l'application n'est pas encore signee.
> Cliquez sur **"Plus d'infos"** puis **"Executer quand meme"** pour continuer.

---

## Acceleration GPU (optionnel)

Si vous avez une carte graphique compatible, vous pouvez accelerer l'IA :

1. Ouvrir **Parametres** dans l'application
2. Aller dans **Acceleration GPU**
3. Choisir votre carte graphique (NVIDIA, AMD, Intel ou Vulkan)
4. Cliquer sur **Telecharger**

Sans carte graphique, l'IA fonctionne normalement sur le processeur (CPU), c'est juste un peu plus lent.
