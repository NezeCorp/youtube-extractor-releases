# YouTube Extractor

Application desktop pour extraire et analyser des videos YouTube avec IA embarquee.

## Telecharger

Rendez-vous dans l'onglet **[Releases](https://github.com/NezeCorp/youtube-extractor-releases/releases)** pour telecharger la derniere version.

## Fonctionnalites

- Extraction de metadonnees YouTube (titres, descriptions, tags, transcriptions)
- IA embarquee (Qwen3-1.7B) - fonctionne 100% hors-ligne
- Multi-fournisseurs IA : Gemini, OpenAI, Groq, Anthropic, Mistral AI, Ollama
- Detection GPU automatique avec acceleration CUDA/ROCm/Vulkan/SYCL
- Gestion de projets et exports (CSV, JSON, Markdown)
- Systeme de mise a jour automatique

## Configuration requise

- Windows 10/11 (x64)
- 4 Go RAM minimum (8 Go recommande)
- ~2 Go d'espace disque
- Connexion internet pour le premier lancement

## Installation

1. Telecharger l'installeur depuis les [Releases](https://github.com/NezeCorp/youtube-extractor-releases/releases)
2. Lancer l'installeur (le modele IA sera telecharge automatiquement ~1.2 Go)
3. L'acceleration GPU peut etre configuree depuis Parametres > Acceleration GPU

> Le Smart Screen Windows peut s'afficher (application non signee) : cliquer "Plus d'infos" puis "Executer quand meme"
