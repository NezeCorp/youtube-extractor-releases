# YouTube Extractor

**Analysez vos vidéos YouTube en quelques clics.**

Extractions, transcriptions, suggestions IA - le tout depuis une seule application.

## C'est quoi ?

YouTube Extractor est une application Windows qui permet de récupérer toutes les informations d'une vidéo ou d'une playlist YouTube : titres, descriptions, tags, miniatures, statistiques et transcriptions.

Une intelligence artificielle intégrée peut ensuite analyser ces données et proposer des améliorations : nouveaux titres, descriptions optimisées, tags pertinents et résumés.

L'IA fonctionne directement sur votre PC, sans envoyer vos données sur internet.

---

## Ce que vous pouvez faire

- **Analyser une vidéo ou une playlist entière** en collant simplement le lien YouTube
- **Récupérer les transcriptions** automatiquement (sous-titres)
- **Obtenir des suggestions IA** pour améliorer vos titres, descriptions et tags
- **Comparer vos contenus** avec les propositions de l'IA côte à côte
- **Organiser vos analyses** par projet
- **Exporter vos données** en CSV, JSON ou Markdown
- **Suivre vos statistiques** : nombre de vidéos analysées, créateurs, historique

---

## L'assistant IA

L'application embarque un modèle d'IA (Qwen3.5-4B) qui tourne entièrement sur votre machine.

- **Aucune connexion internet requise** pour l'IA
- **Aucune clé API nécessaire**
- **Vos données restent privées**

Si vous préférez, vous pouvez aussi connecter un service IA externe : Google Gemini, OpenAI (ChatGPT), Groq, Anthropic (Claude), Mistral AI ou Ollama.

## Installation

1. **Télécharger** l'installeur depuis la page [Releases](https://github.com/NezeCorp/youtube-extractor-releases/releases/latest)
2. **Lancer** le fichier .exe téléchargé
3. **Patienter** pendant l'installation (le modèle IA de ~2.5 Go sera téléchargé automatiquement)
4. **C'est prêt** - lancez YouTube Extractor depuis le menu Démarrer

> **Note :** Windows SmartScreen peut afficher un avertissement car l'application n'est pas encore signée.
> Cliquez sur **"Plus d'infos"** puis **"Exécuter quand même"** pour continuer.

---

## Accélération GPU (optionnel)

Si vous avez une carte graphique compatible, vous pouvez accélérer l'IA :

1. Ouvrir **Paramètres** dans l'application
2. Aller dans **Accélération GPU**
3. Choisir votre carte graphique (NVIDIA, AMD, Intel ou Vulkan)
4. Cliquer sur **Télécharger**

Sans carte graphique, l'IA fonctionne normalement sur le processeur (CPU), c'est juste un peu plus lent.
